# ZX Bank — ATM & Branch Network

## ATM Network

### ZX Bank ATMs
- **Total ATMs**: 8,000+ across India
- **Cash Deposit Machines (CDMs)**: 3,000+ (accept up to 200 notes at once)
- **Recycler ATMs**: 1,500+ (dispense + accept cash)
- **24/7 Operation**: All ATMs available round the clock

### ATM Services
- Cash withdrawal
- Cash deposit (CDM/Recycler)
- Balance enquiry
- Mini statement (last 10 transactions)
- PIN change
- Fund transfer (between ZX Bank accounts)
- Cardless cash withdrawal (via UPI)
- Cheque deposit (at select ATMs)

### Cardless Cash Withdrawal
1. Open ZX Bank App → Cardless Cash
2. Enter amount and beneficiary mobile number
3. Set a temporary code
4. At ATM: Select "Cardless Cash" → Enter mobile number + code
5. Collect cash

### ATM Locator
- **App**: ZX Bank App → ATM Locator (with GPS navigation)
- **Website**: www.zxbank.co.in/atm-locator
- **SMS**: Send ATM <pincode> to 56161
