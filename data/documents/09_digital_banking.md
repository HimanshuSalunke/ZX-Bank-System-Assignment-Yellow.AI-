# ZX Bank — Mobile & Internet Banking

## ZX Bank Mobile App

### Supported Platforms
- Android: 8.0 and above (Google Play Store)
- iOS: 14.0 and above (App Store)

### Key Features
- **Account Management**: View balances, mini-statement, full statement download
- **Fund Transfers**: IMPS, NEFT, RTGS, UPI (instant, 24/7)
- **Bill Payments**: Electricity, gas, water, broadband, DTH, mobile recharges
- **FD/RD Management**: Open, close, premature withdrawal
- **Credit Card**: View statement, pay bills, convert to EMI, redeem rewards
- **Loan Management**: View EMI schedule, prepay, loan statements
- **Investments**: Mutual funds, SIP creation, NPS
- **Demat Services**: View portfolio, IPO applications
- **Insurance**: Buy policies, renew, file claims
- **Card Controls**: Block/unblock, set limits, enable/disable features
- **Cheque Management**: Request cheque book, stop payment
- **Quick Services**: Scan & Pay, QR code payments, collect via UPI

### Registration
1. Download ZX Bank App from Play Store / App Store
2. Enter registered mobile number and debit card details
3. Set MPIN (6 digits) and login password
4. Enable biometric login (fingerprint/face)

### Transaction Limits

| Type | Daily Limit |
|---|---|
| UPI | ₹1,00,000 |
| IMPS | ₹5,00,000 |
| NEFT | ₹10,00,000 |
| RTGS | ₹25,00,000 (min ₹2,00,000) |

## ZX Bank Internet Banking (NetBanking)

### Features
- All features available on mobile app
- Bulk payment uploads (CSV/Excel)
- Tax payment (direct & advance tax)
- E-mandate / SI (Standing Instruction) setup
- Trade finance operations
- Corporate salary uploads
- Account aggregator consent management

### Registration
1. Visit www.zxbank.co.in → NetBanking → New User
2. Enter account number and registered mobile
3. OTP verification
4. Set username and password
5. Optional: Generate virtual keyboard for secure login

### Security Features
- Two-factor authentication (password + OTP)
- Virtual keyboard for password entry
- Session timeout after 5 minutes of inactivity
- Automatic logout on browser close
- Transaction alerts via SMS and email
- IP-based access restrictions (optional)
- Device binding for added security

## UPI Services

### ZX UPI Features
- **UPI ID Format**: mobilenumber@zxbank
- **Transactions**: Send, receive, request money
- **Autopay**: Set up mandates for recurring payments
- **UPI Lite**: Small payments without PIN (up to ₹500)
- **Credit Line on UPI**: Pre-approved credit via UPI (select customers)
- **International UPI**: Send money to Singapore, UAE (select corridors)

### UPI Limits
- Per transaction: ₹1,00,000
- Daily: ₹1,00,000
- UPI Lite per transaction: ₹500
- UPI Lite daily: ₹4,000
